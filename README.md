# UKWOT CRUD Application

This project is a University coursework submission developed as part of a Software Engineering module.

The application is a Django-based web system designed to connect to and interact with a relational SQL database representing operations of a wildlife rescue organisation.

## Objectives

This project aims to achieve:

- Integration of Django with a MySQL relational database  
- Implementation of CRUD (Create, Read, Update, Delete) functionality
